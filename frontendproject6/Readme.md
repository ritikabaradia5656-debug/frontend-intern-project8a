# 📝 React To-Do App

A modern To-Do application built using React and Vite.

## 🚀 Features

- Add tasks
- Delete tasks
- Edit tasks
- Mark as completed
- Filter (All / Completed / Pending)
- Persistent data using localStorage

## 🎨 UI

- Glassmorphism design
- Responsive layout
- Clean modern interface

## 🛠 Tech Stack

- React (Hooks)
- Vite
- CSS

## 📦 Installation

```bash
npm install
npm run dev
